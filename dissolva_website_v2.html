Core Capabilities
Everything you need
in one place.
From raw vessel data to regulatory-ready reports — no Excel macros, no local installs.
Intelligent Compliance Engine
USP/FDA monograph compliance verified at Q-time via interpolation. Instant OOS flagging with academic root-cause analysis.
f1 / f2 Similarity Analysis
FDA-compliant f1 and f2 calculation with 85% cutoff, Q value and Q-time overlays, error bars, and automatic boundary-zone warnings.
62 Kinetic Models
Zero-order to Gompertz, Weibull variants, sigmoid, burst-release, fractal. More models than DDSolver, KinetDS, or RKinetDS.
Excel Report Engine
8-sheet professional reports with logo upload, project metadata, bootstrap results, and dissolution charts — all in one click.
Smart Profile Management
Session persistence across all pages. Profile renaming syncs globally. Project metadata flows into report headers automatically.
Statistical Analysis
MDT, DE, pooled statistics, per-profile tables with SD/RSD. Individual profile plots with error bars and Q-line overlays.
