# DissolvA™ — Website

Bu repo, DissolvA™ platformunun tanıtım web sitesini barındırır.

**Canlı site:** https://USERNAME.github.io/dissolva-website

## Güncelleme nasıl yapılır?

1. `index.html` dosyasını düzenle
2. GitHub'a upload et (Add file → Upload files)
3. 1-2 dakika bekle, site otomatik güncellenir

## Video ekleme

`index.html` içinde şu satırı bulun:
```
src='YOUR_YOUTUBE_EMBED_URL'
```
YouTube video ID'nizi ekleyin:
```
src='https://www.youtube.com/embed/VIDEO_ID'
```

## Uygulama linki

`https://dissanalyze.streamlit.app` yazan yerleri kendi Streamlit URL'iniz ile değiştirin.
